# 📊 Data Visualization Portfolio

I explore questions at the intersection of environment, data, and people—turning complex datasets into clear, actionable insights through visualization and analysis.

This repository showcases my progression in data analysis using R, from foundational graphics to interactive dashboards.

---

## 🔍 Projects

### 📈 ggplot Basics
Introduces core data visualization principles using ggplot2, focusing on clean and effective graphical design.

### 🔄 Data Wrangling (tidyverse)
Covers data cleaning, transformation, and preparation using tidyverse tools.

### ⚖️ Proportions & Distributions
Explores how to represent and interpret proportions and distributions in data.

### 📉 Uncertainty
Visualizes variability and uncertainty using statistical techniques.

### 🔗 Relationships
Analyzes relationships between variables using scatterplots and trend analysis.

### 📊 Comparisons
Focuses on designing visuals that clearly compare groups and categories.

### 📝 Annotations & Storytelling
Enhances plots with annotations to improve clarity and communication.

### 🌐 Interactive Dashboards
Builds interactive visualizations using Plotly and Flexdashboard.
